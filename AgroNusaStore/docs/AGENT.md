# Instruksi Agent AgroNusa

Gunakan bahasa Indonesia untuk semua copywriting web. Prioritaskan kualitas enterprise: desain premium, aksesibilitas, performa, struktur modular, dan naming yang jelas.

## Jika memakai Claude di VS Code

1. Instal ekstensi Claude resmi, Continue, atau Cline yang mendukung Anthropic.
2. Buka pengaturan ekstensi.
3. Pilih provider `Anthropic`.
4. Pilih model Claude yang tersedia di akun kamu.
5. Tambahkan konteks dari file ini dan `.github/copilot-instructions.md`.

Saya tidak dapat mengganti agent VS Code secara langsung dari file proyek karena pengaturan provider berada di ekstensi dan akun VS Code lokal kamu.

## Standar Eksekusi

- Frontend: Next.js, TypeScript strict, Tailwind, component-based.
- Backend: Laravel-style REST API, Repository Pattern, Service Layer.
- UI: bahasa Indonesia, responsive, premium, natural, high conversion.
- Data: gunakan istilah pertanian nyata, bukan lorem ipsum.
- Keamanan: validasi input, JWT, policy authorization, rate limit API.
